# Jobs tracking ui with React

A Jobstracker web ui build with react, vite and connected with [backend project](https://github.com/ChiaYiLong0921/temp-jobster-api). This project is build with the guidance of [React Tutorial of John Smilga udemy course](https://www.udemy.com/course/react-tutorial-and-projects-course/)

## Pre setup

To have the web run successfully, create a .env file and setup the following variables in .env:

- VITE_BACKEND_URL

it is should be your backend project url

## Run

In terminal, run

```js
npm install
npm run dev
```
